# 鸣潮·挑战配队顾问

## 何时使用

用户消息匹配以下意图时启用：

- 「本期 / 这期 / 当前 深塔 / 逆境深塔 / 深境区 怎么打 / 怎么配队 / 用啥角色」
- 「本期 / 这期 / 当前 海墟 / 冥歌海墟 / 海隙 / 湍渊 / 无尽 怎么配 / 阵容」
- 「本期 / 这期 / 当前 矩阵 / 矩阵叠兵 / 稳态协议 / 奇点扩张 阵容 / 配队 / 怎么打」
- 「帮我分析下当期挑战 / 三大挑战 / 周期挑战」

不要为「角色培养建议 / 单角色配队 / 历史期数」启用此 skill；那些走常规
`get_char_detail` / `filter_chars_wuwa` 即可。

## 关卡基础规则（官方名称 + 机制）

> 下面三块都是 **周期性更新**：每期 buff、敌人、关卡布局都会变，所以
> 必须先调 `get_current_period_wuwa(...)` 拿本期数据，不要凭印象答。

### 逆境深塔（Tower of Adversity）

- 三个区：**稳定区** / **实验区** / **深境区**（旧称"超载区"）。
- **深境区**就是本 skill 主要关心的"周期挑战"，分**左塔（残响之塔）/
  中塔（深境之塔）/ 右塔（回音之塔）**三座，每座 4 层共 **12 个挑战位**。
- 每个角色每期共有 **10 点疲劳值**，挑战不同层消耗对应点数，耗完该角色
  本期就不能再上场。角色在不同塔之间也算同一份疲劳。
- **难度集中在第 4 层**：左塔/右塔的 1-3 层都很轻松，任意能用的队就能
  过；真正需要排兵布阵的是 **左塔第 4 层 + 中塔全部 4 层 + 右塔第 4 层**，
  共 6 个"硬位"。
- 每塔通常有定向 buff（如"敌方湮灭/热熔抗性降低"或"变奏技额外触发热熔
  范围伤害"等）和奖励星声，深境区每期满星 750 星声。

### 冥歌海墟（Mournful Song Abyss）

- 永久挑战，分**再生海域·海隙**和**再生海域·湍渊**两条副本线，会按期
  周期性重置。
- **配队顾问只关心"无尽"部分**——也就是深阶最高计分挑战，输出多寡
  决定排行。其余层（中阶 7-8 层无抗性 / 深阶 9-11 层有抗性）按本期 buff
  随手打就行，不要为它们写详细方案。
- 2.6 版本起的 **焚烬系统**：战斗中累积焚烬值，攒满后激活强力 **焚潮**
  增伤效果。本期 buff 通常和"焚烬触发条件 / 焚潮持续期收益"挂钩。
- 无尽是 3 人队，要拿出最高战 + 最契合本期焚潮 buff 的组合。

### 矩阵叠兵（Matrix Stratagem，**注意：不是"残象矩阵"**）

- 3.0 版本上线的全新周期挑战，分两个部分：

  1. **稳态协议**：固定 4 个 BOSS，每队 3 人，**队伍数量没有硬性
     上限**（不要假设"最多 3 队"这种过时说法）。**奶妈/治疗类角色
     （维里奈、白芷、守岸人 等）整个稳态协议最多出场 2 次**，其他
     角色没有这个出场次数限制——所以多 BOSS 战里必须把奶妈安排到
     最需要奶量的两场 BOSS，剩下场次给纯输出/带护盾的辅助。**接力
     机制**：上一队没在时间内击败 BOSS，下一队继承 BOSS 剩余血量
     继续打。
  2. **奇点扩张**：无尽模式，循环刷新 4 只怪物，每轮血量上升 + 解锁更强
     词条加成。上场队伍数 **无限**，目标是综合最高分。

- 解锁要求：深境区累计 ≥ 24 印记 **且** 再生海墟·湍流（湍渊）累计积分
  ≥ 3500。

> 概念区分：**残象聚落** ≠ 矩阵叠兵。残象聚落是声骸 farming 玩法，跟
> 配队顾问无关，不要混淆。

## 工作流（严格按顺序执行）

> ⚠️ 所有 tool 调用都是只读查询，不需要用户确认；但要把握 token 预算，
> 同一 tool 同一参数一次工作流内只调一次。

### Step 1 — 拉本期信息

先拿当期期数 + 起止时间：

- 深塔（深境区）： `get_current_period_wuwa(mode="tower")`
- 海墟（海隙 / 湍渊）： `get_current_period_wuwa(mode="slash")`
- 矩阵叠兵： `get_current_period_wuwa(mode="matrix")`
- 用户只说"本期挑战 / 三大挑战"，三个都拉。

`get_current_period_wuwa` 只给出期数和起止时间——**详细的 buff、敌人 monster_id、
层级编号都在对应 KP 里**，必须**再调** `search_wuwa_kb(query=...)` 取详情：

- 深塔： `search_wuwa_kb(query="鸣潮深塔第N期")` → ww_tower_N
- 海墟： `search_wuwa_kb(query="鸣潮海墟第N期")` → ww_slash_N
- 矩阵： `search_wuwa_kb(query="鸣潮矩阵第N期")` → ww_matrix_N

`search_wuwa_kb` 返回的 `content` 字段里就有完整的 buff 描述、怪物名/ID、各塔/区/位
的具体内容。

### Step 2 — 查敌人抗性

对 Step 1 列出的每个 `monster_id`：

- `get_monster_resistance_wuwa(monster_id)` 拿到属性抗性表（冷凝/热熔/导电/气动/衍射/湮灭）。
- 抗性 ≥ 0.4 视为"该属性输出 DPS 不推荐"。
- 抗性 ≤ -0.2 视为"该属性强势克制"。

可选： `recommend_against_monster_wuwa(monster_id)` 让后端直接给"克制属性 +
推荐角色"雏形，作为 Step 4 的候选起点（但仍要结合 buff 和用户练度过一遍）。

### Step 3 — 解读 buff 适配什么角色

把 Step 1 拿到的 buff 文本读一遍，提炼：

- buff 是哪种**触发条件**（共鸣回路 / 普攻 / 重击 / 协同攻击 / 变奏技 /
  焚烬激活 / 元素反应 等）
- buff 是哪种**收益形式**（伤害提升% / 攻击力% / 特定技能倍率 /
  焚潮增伤 等）
- buff 锁定哪个**属性**或哪类**武器/共鸣**（如果有）

把这些条件翻译成"哪类角色吃得到"。

### Step 4 — 看用户练度

调用：

- `get_user_wuwa_uids(user_id, bot_id)` 拿到用户绑定的鸣潮 UID 列表。
- **只用国服 UID**（9 位、`1xxxxxxxx` 起头）。国际服 UID（`8xxxxxxx` /
  以 8 开头）忽略。
- 如果用户名下没有任何国服 UID：**跳过 Step 4**，按"通用强角 + 本期
  buff"出方案，并在最终输出里明确提醒"未检测到国服 UID 绑定，方案
  基于通用强角假设，建议用 `xwbd <UID>` 绑定后重问以获得练度匹配建议"。
- 有国服 UID 时：
  - `get_user_wuwa_char_list(uid)` 看持有角色 + 命座/武器/等级。
  - `get_user_wuwa_char_scores(uid)` 看声骸打分，挑高分角色优先入队。

### Step 5 — 撮合方案

按"克制属性 + buff 吃满 + 用户已有高练度"三重打分挑队伍：

- **深塔·深境区**：左/中/右三塔各 4 层共 12 个位。**重点配队只需关注
  左塔第 4 层、中塔全部 4 层、右塔第 4 层**——左右两塔的前 3 层难度
  很低，几乎任何练度的队都能过，不要为它们写详细方案；这些层提一句
  "前 3 层练度够即可，建议把高战角色和疲劳留给第 4 层"就行。每个角色
  10 点疲劳，跨塔共享，所以中塔 4 层是主要的疲劳消耗大户。
- **冥歌海墟**：只输出"无尽"部分的方案——最高战 + 最契合本期焚潮 buff
  的 3 人队。中阶/深阶其余层不要写详细配队，提一句"按本期 buff 顺手打
  就行"即可。
- **矩阵叠兵·稳态协议**：4 BOSS，每队 3 人，**队伍数量没有硬上限**。
  **奶妈类角色（维里奈/白芷/守岸人 等）整个稳态协议最多出场 2 次**——
  必须挑「需奶量最大的 2 场 BOSS」放奶妈，剩下两场用纯输出/带护盾的
  辅助顶上（或干脆选续航能力强的 C 位走"不带奶"流派）。利用接力
  机制：让爆发队先打，残血给稳输出队收。
- **矩阵叠兵·奇点扩张**：上场队数无限——把所有合格 3 人队都列出（建议
  4-6 队），按总分排序，说明每队的 C / 子 C / 辅 / 用到的本期词条。

候选角色不够时：用 `filter_chars_wuwa(attribute=..., role="主C/副C/治疗/辅助")`
拓展全角色池作为"可培养目标"建议（标注用户尚未拥有）。

### Step 6 — 输出格式

用人话给用户讲，不要丢一大堆 raw JSON。最少包含：

1. **本期一句话总结**（期数 + 主敌人抗性侧重 + buff 主题）
2. **每个挑战的队伍方案**：列出塔/位/队伍编号、3 名角色（C/副C/辅）、
   为什么这么配（buff 命中 + 克制 + 用户高练度）
3. **不足提醒**：用户缺什么关键角色 / 哪些位只能用替补，建议练谁

如果用户问的是单一挑战（比如"深塔"），只输出深塔部分。

## 注意

- 角色名一律用官译中文（如「珂莱塔」「卡提希娅」），不要 raw `r_n`。
- 鸣潮抽卡叫**唤取**，不是抽卡/调谐（"调谐"只在 banner 池名里）。
- buff 数值不要随便猜——只引用 Step 1 拿到的原文。
- 队伍输出后简短解释，不要列出每条 buff 的全文。
